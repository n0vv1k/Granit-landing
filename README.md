# Granit Landing Page

Простой landing page для компании по продаже и установке гранитных изделий.
Проект представляет собой одностраничный сайт с адаптацией под мобильные устройства.

## 📌 О проекте

Landing page создан для демонстрации услуг компании и привлечения клиентов.

Основные блоки страницы:

* Главный экран (Hero)
* О компании
* Преимущества
* Услуги
* Контактная форма
* Footer

## ⚙️ Используемые технологии

* **HTML5**
* **CSS3**
* **JavaScript**

## 📱 Адаптивность

Сайт адаптирован под мобильные устройства с шириной экрана **≈380px**.

Для адаптации используются:

* Flexbox
* Media queries
* Относительные размеры элементов

## 🚀 Запуск проекта

1. Скачайте или клонируйте репозиторий

```
git clone https://github.com/your-username/granit-landing.git
```

2. Откройте папку проекта

3. Запустите файл **index.html** в браузере

Также можно использовать локальный сервер (например **Live Server** или **OpenServer**).
